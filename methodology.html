Methodology

How the index is built

From counting places to grouping neighborhoods, in plain language. Five steps and a few honest caveats.

1

Count the places

The index maps every kind of place a child can reach books, and weights each by how directly it puts a book in a child's hands.

Museums count 0.5, except children's museums, which count 1. Botanical gardens and zoos are not included.

Schools are weighted by staffing. A school library counts 1 if no librarian is on staff, climbing to 6 as the student-to-librarian ratio improves.

A public library counts ten times a Little Free Library, because it's free, permanent, staffed, and built for reading.

2

Measure how far

For every neighborhood we measure how long it takes to reach each place, by car and by public transport (bus and rail), and a place's value fades the longer it takes to get there.

Technical: an exponential travel-time decay with β ≈ 0.08 per minute. A place about 9 minutes away counts about half as much as one next door.

3

Add it up

A neighborhood's score is the sum, over every reachable place, of its weight times how close it is. More places, and closer or higher-weighted places, both raise the score.

The same neighborhood, scored two ways: strong by car, thin by transit. That gap is what the index is built to measure.

4

Sort into easy categories (bins)

Raw scores are hard to read, so we rank every neighborhood into five equal-size groups (Very low, Low, Moderate, High, Very high), and we do it separately for car and for transit, so each map reads clearly on its own.

Transit has an extra "No transit access" group for neighborhoods nothing reaches by bus or rail. Unpopulated areas (no resident children) are set aside entirely.

An honest caveat: because the two modes are binned separately, a transit "High" is high for transit. In absolute terms a car still reaches far more, almost everywhere.

One more caveat: because groups are ranked against the current statewide data, a neighborhood's group can shift when other neighborhoods change, not just when it does. These groups compare places to each other today. They are not built to track improvement over time. Tracking change is a separate exercise, done on the underlying scores.

5

Step 5, Does reaching books depend on a car?

The index values the whole set of book sources a neighborhood can reach, weighted by type and faded by travel time. It is not the trip to the nearest book. So we read each neighborhood in two steps.

An honest anchor. Transit tiers are relative to transit across Connecticut, which is modest statewide. Even strong transit access reaches a narrower set of book sources than a car does. The point is not that transit matches a car. It is that some neighborhoods give a car-free family a real option, and others leave them with almost none.

A suburb on a rail line with local bus reads as strong transit access. A dense town center reads good to strong, depending on routes. A large-lot exurb with roads but few buses reads limited, mostly needs a car. A remote rural town reads as few or very few sources to begin with.

For district views we average each neighborhood's score by where children live, then route the district through the same two-question tree. Averaging can hide contrast inside large districts. A district holding both a transit-served center and car-dependent outskirts will read as middling, even though few of its neighborhoods are actually middling. The neighborhood view is the honest grain; the district view is the summary.

What the index can't tell you

  • It doesn't see access just across the state border, even when that's the closest library.
  • It counts schools as reachable for every nearby child, even though a child attends only their own.
  • Transit bins are relative to transit, not absolute. See step 4.
  • It's one lens on book access, not a ranking of communities.

The index is refreshed as data updates.
